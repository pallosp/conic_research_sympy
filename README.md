# Calculating with conics (research)

## Installation

```
git clone https://github.com/pallosp/conic_research_sympy
python3 -m venv .venv
source .venv/bin/activate
pip install -e '.[dev]'
```

## API reference

[API reference](docs/api.md)

To regenerate the docs, run

```
pip install -e '.[docs]'
pydoc-markdown
```
